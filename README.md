# pqdi-carnet-de-contacts-fil-rouge
Présentation du projet fil rouge pour la promotion 2026-2027
🔐 Carnet de Contacts Sécurisé (Python)
📌 Projet fil rouge – Formation PQDI
🚀 Présentation

Ce projet est une application console développée en Python dans le cadre d’un parcours de pré-qualification en développement informatique.

L’objectif est de créer un carnet de contacts sécurisé, permettant la gestion complète de contacts avec sauvegarde locale et chiffrement des données.

🎯 Objectifs pédagogiques

Ce projet permet de travailler :

algorithmique de base
programmation Python
structuration d’un programme
gestion de fichiers
notions de sécurité (chiffrement simple)
utilisation de Git et travail collaboratif
introduction à l’IA comme assistant de développement
🧩 Fonctionnalités

L’application permet de :

✔ Ajouter un contact
✔ Afficher tous les contacts
✔ Rechercher un contact
✔ Modifier un contact
✔ Supprimer un contact
✔ Sauvegarder les données sur disque
✔ Chiffrer les données avant stockage

📇 Structure d’un contact

Un contact contient :

nom
prénom
téléphone
email
💻 Technologies utilisées
Python 3
JSON / fichiers texte
Algorithmes simples de chiffrement
Git (versionnement)
